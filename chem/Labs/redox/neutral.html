Example: Oxidation numbers of neutral atoms

Noble gasses are a good example of atoms which do not combine to form compounds. As such, noble gasses are almost always found in nature as neutral atoms. Any pure element is given an oxidation number of 0.

The oxidation number on Ar is 0.
However, even reactive elements such as sodium can be found in reactions in their pure elemental form. Note that this means that the atom has not formed a compound. An atom which is found in a compound may have an oxidation number which is not 0. Any atom which esists in a compound as an ion will definitely have a non-zero oxidation number.

The oxidation number on Na is
0
not 0
The oxidation number on K in K2SO4 is
0
not 0

Molecular Elements

Many elements do not appear as atoms in their pure elemental form. Common examples of such elements are oxygen, phosphorus, chlorine and sulfur. However, because these species are composed of only one element, the atoms are given an oxidation number of zero.

The oxidation number on O in O2 is
0
not 0
The oxidation number on P in P4 is

not 0
The oxidation number on Cl in Cl2 is
0
not 0
The oxidation number on S in S8 is
0
not 0
The oxidation number on S in H2SO4 is
0
not 0
The oxidation number on O in H2O is
0
not 0
